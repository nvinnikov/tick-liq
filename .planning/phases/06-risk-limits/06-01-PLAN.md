pub struct RiskMonitor {

}

**`RiskMonitor::new(state, max_drawdown_pct, max_il_pct, drift_min_margin_ratio)` constructor.**

**`evaluate(&mut self, snap: &crate::storage::writer::PnlSnapshot, drift_margin_ratio: Option<f64>) -> RiskAction` method:**

The `drift_margin_ratio` is passed as `Option<f64>` -- the caller fetches it externally. This keeps the evaluate() method synchronous and testable without RPC mocking.

Evaluation order (per D-05, D-06):
1. If `self.state.halt_flag` is true, return `HaltAll { drawdown_pct: self.state.current_drawdown_pct }` immediately.
2. Update peak_pnl high-water mark: `if snap.net_pnl > self.state.peak_pnl { self.state.peak_pnl = snap.net_pnl; }`
3. Drawdown check (per D-10, D-11): if `max_drawdown_pct` is Some AND `self.state.peak_pnl > 0.0`, compute `drawdown_pct = (peak_pnl - snap.net_pnl) / peak_pnl * 100.0`. Update `self.state.current_drawdown_pct = drawdown_pct`. If `drawdown_pct > max_drawdown`, set `self.state.halt_flag = true`, return `HaltAll { drawdown_pct }`.
4. IL check (per D-07, D-08, D-09): compute `il_pct = if snap.position_value > 0.0 { snap.il_usd.abs() / snap.position_value * 100.0 } else { 0.0 }`. If `max_il_pct` is Some:
   - If `il_pct > max_il` and NOT already paused: set `self.state.pause_flag = true`, return `PauseRebalancing { il_pct }`.
   - If `il_pct <= max_il` and IS paused: set `self.state.pause_flag = false`, return `ResumeRebalancing { il_pct }`.
   - If `il_pct > max_il` and IS already paused: return `PauseRebalancing { il_pct }` (propagate).
5. Drift margin check: if `drift_min_margin_ratio` is Some and `drift_margin_ratio` is Some(ratio), and `ratio < threshold`, return `CloseDriftHedge { margin_ratio: ratio }`.
6. Return `Continue`.

Update `self.state.updated_at = chrono::Utc::now()` before returning any action.
