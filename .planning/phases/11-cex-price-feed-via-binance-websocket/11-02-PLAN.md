## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User → CLI | `--cex-symbol` value is user-supplied text, passed verbatim into URL construction. |
| cex_ws task → shared state | Writer from tokio-spawned async task, reader from sync `on_notify` closure on a tokio worker thread. |
| Shared state → downstream consumers | `price_current: f64` flows into `compute_il`, `PnlSnapshot.price`, and the DB `pnl_history` insert. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-10 | Tampering | User-supplied `--cex-symbol` | mitigate | Symbol is forced to lowercase before URL construction (`build_stream_url` in Plan 01); any invalid symbol yields HTTP 400 on handshake and triggers reconnect loop — does NOT inject into other components.
